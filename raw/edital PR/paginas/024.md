### 16.5. Termo de Entrega da Register Key

O contratado deverá assinar termo contendo:

- Software entregue;
- Versão;
- Licença;
- Chave;
- Escopo;
- Ambientes;
- Data;
- Responsável técnico.

### 16.6. Critérios de Aceite

A entrega será considerada aceita quando:

- ✓ software instalado
- ✓ register key válida
- ✓ licença compatível
- ✓ módulos ativos
- ✓ acesso funcional
- ✓ documentação entregue
- ✓ teste realizado

### 16.7. Não Conformidades

Serão consideradas falhas:

- Licença inválida;
- Chave incorreta;
- Restrição de uso;
- Limitação não contratada;
- Expiração indevida;
- Bloqueio de funcionalidades.

Prazo de correção: 7 dias (úteis).

### 16.8. Responsabilidade do Contratado

O contratado garante que:

- Possui direitos de licenciamento;
- A licença é legal;
- Não há violação de propriedade intelectual;
- A chave permite uso integral contratado;
- A licença permanecerá válida durante o contrato.

### 16.9. Renovação / Substituição de Chave

Em caso de:

- Renovação;
- Upgrade;
- Mudança de ambiente;
- Troca de servidor;
- Ampliação de usuários;
- O contratado fornecerá nova register key sem ônus adicional quando previsto contratualmente.
